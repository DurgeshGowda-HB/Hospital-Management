# Hospital Management 

A web-based Hospital Management System built using the MERN Stack (MongoDB, Express.js, React.js, and Node.js). This project helps manage hospital operations digitally by providing separate modules for Admin, Doctors, and Patients.

## Features

* User Registration and Login
* JWT Authentication
* Admin Dashboard
* Patient Registration
* Doctor Management
* Appointment Booking
* Patient Records Management
* Profile Management
* CRUD Operations
* Responsive UI

## Modules

### Admin

* Manage Doctors
* Manage Patients
* Manage Appointments
* Dashboard Analytics

### Doctor

* View Appointments
* Update Patient Records
* Manage Availability

### Patient

* Register / Login
* Book Appointments
* View Medical Records
* Update Profile

## Tech Stack

* Frontend: React.js
* Backend: Node.js + Express.js
* Database: MongoDB
* Authentication: JWT
